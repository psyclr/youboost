# YouBoost — Гайд для пользователя

## Что такое YouBoost?

YouBoost — это маркетплейс SMM-услуг. Ты заходишь, пополняешь баланс, выбираешь услугу
(например, "5000 просмотров на YouTube"), вставляешь ссылку на видео — и система
всё делает за тебя. Когда заказ выполнен, приходит уведомление на почту.

---

## Регистрация и вход

### Регистрация

Заходишь на страницу регистрации, заполняешь:

| Поле     | Требования                                                            |
| -------- | --------------------------------------------------------------------- |
| Email    | Валидный адрес, не занят другим аккаунтом                             |
| Username | 3-30 символов, буквы/цифры/подчёркивание                              |
| Пароль   | Минимум 8 символов, хотя бы одна заглавная, одна строчная, одна цифра |

После регистрации автоматически создаётся кошелёк с нулевым балансом.

### Вход

Вводишь email + пароль. Система выдаёт тебе токен (действует 1 час).
Когда час истекает — токен обновляется автоматически, тебя не выкидывает.

### Выход

При выходе все токены отзываются. Даже если кто-то перехватил твой токен — он перестанет работать.

## Кошелёк и деньги

### Три цифры на странице баланса

```
Balance:   $50.00   ← Сколько всего денег на счету
Frozen:    $10.00   ← Заморожено под активные заказы
Available: $40.00   ← Сколько можно потратить прямо сейчас
```

**Frozen** — это не списание. Когда ты создаёшь заказ на $10, деньги замораживаются.
Если заказ выполнен — они списываются. Если провален — возвращаются обратно в Available.

### Пополнение баланса

1. Жмёшь "Deposit" на странице биллинга
2. Указываешь сумму (в USD) и криптовалюту (USDT)
3. Система генерирует **платёжный адрес** и **QR-код**
4. Отправляешь крипту на этот адрес
5. Подтверждаешь оплату в интерфейсе (вводишь txHash транзакции)
6. Деньги зачисляются на баланс

Каждый депозит имеет **срок действия** — если не оплатить вовремя, он просрочится.

### История транзакций

На странице `/billing/transactions` видно все операции:

| Тип     | Что значит                          |
| ------- | ----------------------------------- |
| DEPOSIT | Пополнение баланса                  |
| HOLD    | Заморозка под заказ                 |
| RELEASE | Разморозка (заказ отменён/провален) |
| CHARGE  | Списание (заказ выполнен)           |
| REFUND  | Возврат от админа                   |

Каждая запись хранит баланс "до" и "после" — полная прозрачность.

## Каталог услуг

Страница `/catalog` — публичная витрина. Доступна даже без регистрации.

### Что видно в каталоге

Каждая услуга показывает:

- **Название**: "YouTube Views — Premium"
- **Платформа**: YouTube, Instagram, TikTok, Twitter, Facebook
- **Тип**: Просмотры, Подписчики, Лайки, Комментарии, Репосты
- **Цена за 1000 штук**: например $1.50
- **Мин/макс количество**: от 100 до 100000

### Фильтры

Можно фильтровать по платформе и типу. Например: "Покажи только лайки на Instagram".

## Создание заказа

### Шаг за шагом

1. Выбери услугу из каталога
2. Вставь ссылку на контент (например, `https://youtube.com/watch?v=...`)
3. Укажи количество (в рамках мин/макс для этой услуги)
4. Система покажет итоговую цену: `количество x цена_за_1000 / 1000`
5. Нажми "Заказать"

### Что происходит после нажатия

```
Ты нажал "Заказать"
    |
    v
Система проверяет: хватает ли денег?
    |
    v
Деньги замораживаются (не списываются!)
    |
    v
Заказ отправляется поставщику
    |
    v
Статус: PROCESSING — ждём выполнения
```

### Расчёт цены

Формула простая:

```
Цена = (количество x цена_за_1000) / 1000

Пример: 5000 просмотров x $1.50 за 1000 = $7.50
```

## Статусы заказов

| Статус     | Что значит                        | Деньги                              |
| ---------- | --------------------------------- | ----------------------------------- |
| PENDING    | Заказ создан, ждёт отправки       | Заморожены                          |
| PROCESSING | Отправлен поставщику, выполняется | Заморожены                          |
| COMPLETED  | Всё сделано                       | Списаны                             |
| PARTIAL    | Сделано частично (3000 из 5000)   | Частично списаны, остаток возвращён |
| FAILED     | Не удалось выполнить              | Возвращены полностью                |
| CANCELLED  | Ты отменил заказ                  | Возвращены полностью                |
| REFUNDED   | Админ сделал возврат              | Возвращены полностью                |

### Пример PARTIAL

Заказал 5000 просмотров за $7.50. Поставщик смог доставить только 3000 (60%).

- Списано: $4.50 (60% от $7.50)
- Возвращено: $3.00 (40% от $7.50)

### Отмена заказа

Отменить можно только заказы со статусом PENDING или PROCESSING.
Деньги возвращаются на баланс моментально.

## API-ключи (для разработчиков)

Если хочешь работать с YouBoost программно (из своего кода), можно создать API-ключ.

### Как создать

1. Зайди на `/api-keys`
2. Нажми "Generate Key"
3. Скопируй ключ — **он показывается только один раз!**

Формат ключа: `yb_a1b2c3d4e5f6...` (64 символа после `yb_`)

### Как использовать

Добавляй в HTTP-запросы заголовок:

```
Authorization: Bearer yb_твой_ключ_сюда
```

### Лимиты

| Тариф      | Запросов в минуту |
| ---------- | ----------------- |
| BASIC      | 100               |
| PRO        | 500               |
| ENTERPRISE | 2000              |

### Управление

- Можно создать несколько ключей
- Можно отозвать ключ (он перестанет работать)
- Можно задать срок действия (опционально)
- Видно когда ключ последний раз использовался

## Вебхуки (уведомления на твой сервер)

Вебхуки — это способ получать уведомления о событиях на **свой URL**.
Вместо того чтобы проверять статус заказа вручную, YouBoost сам отправит POST-запрос
на указанный тобой адрес, когда что-то произойдёт.

### Как настроить

1. Зайди на `/webhooks`
2. Нажми "Create Webhook"
3. Укажи:
   - **URL**: `https://my-server.com/youboost-hook`
   - **Events**: какие события слушать (order.created, order.completed, и т.д.)
   - **Secret**: секретный ключ для подписи (система сгенерирует)

### Какие события доступны

| Событие         | Когда срабатывает                   |
| --------------- | ----------------------------------- |
| order.created   | Заказ создан и отправлен поставщику |
| order.completed | Заказ полностью выполнен            |
| order.partial   | Заказ выполнен частично             |
| order.failed    | Заказ провалился                    |
| order.cancelled | Заказ отменён                       |

### Что приходит на твой URL

```json
{
  "event": "order.completed",
  "data": {
    "orderId": "abc-123",
    "status": "COMPLETED",
    "remains": 0
  },
  "timestamp": "2026-02-26T12:00:00Z"
}
```

Плюс заголовки:

- `X-Webhook-Signature` — HMAC-SHA256 подпись (проверяй для безопасности)
- `X-Webhook-Event` — название события

### Надёжность

Если твой сервер не ответил или упал — YouBoost **повторит доставку 3 раза**
с нарастающими интервалами (10 сек, 20 сек, 40 сек).

## Email-уведомления

Приходят автоматически при каждом важном событии:

- Заказ создан
- Заказ выполнен
- Заказ провален
- Заказ частично выполнен
- Заказ отменён (с суммой возврата)

Отключить пока нельзя — приходят всегда.

---

## Безопасность

- Пароль хешируется и никогда не хранится в открытом виде
- API-ключи хранятся как SHA256-хеши (даже при утечке базы ключ не восстановить)
- Вебхуки подписаны HMAC-SHA256 — всегда проверяй подпись на своей стороне
- Все финансовые операции атомарные (деньги не могут "потеряться" между операциями)
- Полный аудит-трейл: каждая транзакция записана в ledger с балансом до/после
